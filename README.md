# OSA Consulting - Sitio Web
Este es el repositorio para el sitio web oficial de OSA Consulting, una consultora especializada en Revenue Operations (RevOps).

## Getting Started

1. Install Dependencies
```
npm i
```
3. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
